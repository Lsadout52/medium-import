2. Creating an IAM User

  1. Navigate to IAM Console in AWS.
  2. Click on Users > Add User.
  3. Enter a username and choose the type of access:
    • AWS Management Console (for UI access)
    • Programmatic Access (for CLI/API access)
  4. Assign the user to an IAM Group or attach permissions directly.
  5. Review settings and Create User.

3. Creating an IAM Group

  1. In the IAM Console, go to Groups and click Create Group.
  2. Enter a Group Name (e.g., Developers, Admins).
  3. Attach necessary policies (e.g., AdministratorAccess, ReadOnlyAccess).
  4. Add existing IAM Users to the group.
  5. Click Create Group.

Now, all users in the group inherit its permissions, making it easier to manage access across teams.
