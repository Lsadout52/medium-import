When launching EC2 instances or configuring networking in AWS, you will encounter Private IPs, Public IPs, and Elastic IPs. Understanding how they work is essential for managing connectivity and security.

1. Private IP

A Private IP is an internal address assigned to an instance inside a VPC. It allows communication between instances within the same network but cannot be accessed from the internet.
